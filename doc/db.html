基本使用

数据库虽然只是功能驱动的一种，但也是使用最频繁的功能之一，相对比较重要，所以用单独的章节来介绍。

数据库驱动

驱动 描述
Mysqli 基于php mysqli扩展，支持一些特有的mysql方法
Mysql 基于php pdo_mysql扩展
Pgsql 基于php pdo_pgsql扩展（粗测）
Sqlite 基于php pdo_sqlite扩展（粗测）
Sqlsrv 在win系统下使用pdo_sqlsrv扩展，类unix系统下使用pdo_odbc扩展（未测试）
Oracle 基于php pdo_oci扩展（未测试）
Cluster 支持设置多个数据库服务器，实现读写分离主从分离，原理是根据SQL的SELECT INSERT等语句将请求分配到不同的服务器。

使用SQL

增改查删

exec方法

执行一条SQL，并根据语句类型返回不同结果

INSERT：返回插入纪录的insertId，如果没有insertId则返回true

UPDATE：返回更新的行数affectedRows

DELETE：返回删除的行数affectedRows

SELECT：返回获取的所有数据fetchAll

query方法

执行一条SQL，返回query由后续方法处理结果，如fetch fetchRow fetchAll numRows affectedRows insertId等方法
